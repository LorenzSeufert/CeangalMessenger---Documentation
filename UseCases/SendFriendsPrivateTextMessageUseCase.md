# 2 Flow of Events

## 2.1 Basic Flow

- User clicks on "Friends" button
- Redirect to "Friends" page
- A list of all friends is shown
- User selects a friend
- Chat window opens with the friend
- User types message into input field
- User clicks "Enter"
- Message get send to friend
